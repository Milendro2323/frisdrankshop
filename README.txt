Installatie
1) Maak database 'frisdrankshop' in phpMyAdmin.
2) Importeer fris.sql in die database.
3) Plaats deze map in C:\xampp\htdocs\frisdrankshop
4) Start Apache en MySQL in XAMPP.
5) Open http://localhost/frisdrankshop/?page=home
